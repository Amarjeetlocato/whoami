## üîß Easy Customization

### Change Colors
Edit in `assets/css/main.css`:
```css
:root {
  --accent: #2563eb;    /* Primary color */
  --accent-2: #f97316;  /* Secondary color */
  --accent-3: #0891b2;  /* Tertiary color */
}
```

### Update Text
Edit directly in HTML files - no build process needed.

### Add New Section
1. Copy existing section HTML
2. Update content
3. Use existing CSS classes
4. Maintain responsive grid pattern

### Add New Product
1. Add card in `products.html`
2. Update `assets/data/products.json`
3. Use existing product card styling
